<threat_model>
## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| Internet → Nginx (CloudPanel) | Terminació TLS i upgrade WebSocket; entrada no confiable des d'Internet |
| Nginx → Node (loopback 127.0.0.1:8011) | Trànsit pla en loopback; l'app mai veu TLS |
| Client admin → Socket.io | El professor prova el rol admin amb ADMIN_SECRET; qualsevol client podria reclamar-lo si el secret és absent |
| Fitxer `.env` al VPS | Magatzem del secret; mai ha d'entrar al repo |

## STRIDE Threat Register

| Threat ID | Category | Component | Severity | Disposition | Mitigation Plan |
|-----------|----------|-----------|----------|-------------|-----------------|
| T-05-02 | Elevation of Privilege | `ADMIN_SECRET` absent després d'un restart de PM2 → auth admin deshabilitada (T-04.1-05 transferit) | high | mitigate | `.env` al VPS proporciona ADMIN_SECRET; dotenv el carrega abans que registerSocketHandlers el llegeixi; verificar que pm2 logs NO mostra el warning (Task 3) |
| T-05-03 | Information Disclosure / Tampering | Transport sense xifrar | medium | mitigate | HTTPS-only + redirecció HTTP→HTTPS via CloudPanel/Let's Encrypt (D-08); el vhost força https |
| T-05-05 | Denial of Service (availability) | Vhost reverse-proxy sense directives d'upgrade → app websocket-only falla del tot | high | mitigate | Verificar/afegir `proxy_http_version 1.1` + `Upgrade`/`Connection` al vhost i confirmar `101` al Network tab (Task 2/3, D-06/D-07) |
</threat_model>
