<threat_model>
## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| PC d'equip (navegador) → servidor Socket.io | Entrada no confiada: intents `team:select`, i potencialment `admin:*` forjats des de devtools |
| PC admin (navegador) → servidor Socket.io | Es declara admin via `handshake.auth.role`; sense login (risc residual acceptat) |
| localStorage del client → servidor (token) | El token torna al servidor a cada handshake; és l'única prova d'identitat d'equip |

## STRIDE Threat Register

| Threat ID | Category | Component | Severity | Disposition | Mitigation Plan |
|-----------|----------|-----------|----------|-------------|-----------------|
| T-01-01 | Elevation of Privilege | handlers `admin:*` a socketHandlers.js | high | mitigate | Guard server-side `socket.rooms.has('admin')` a cada handler admin; només sockets amb `handshake.auth.role==='admin'` s'uneixen al room admin. Un socket d'equip mai pot mutar l'estat (Test E). |
| T-01-02 | Spoofing | sessionStore token | medium | mitigate | Tokens via `crypto.randomUUID()` (CSPRNG); mai `Math.random()`; mai exposats a `getPublicState()`. |
| T-01-03 | Tampering (XSS) | render de noms d'equip a admin.js | medium | mitigate | Inserció via `textContent` exclusivament (grep == 0 `innerHTML` a client); sense camps rich-text a la Fase 1. |
| T-01-04 | Denial of Service | handlers de socket | medium | mitigate | Validació de forma de payload (array de strings, longitud) + embolcall try/catch per handler perquè una excepció no tombi el procés compartit. |
| T-01-05 | Info Disclosure / EoP | ruta estàtica `admin.html` | high | accept | Cap PC d'equip pot mutar l'estat sense ser al room admin (T-01-01). El risc residual (un PC en blanc navega a admin.html i es declara admin) s'accepta per l'abast sense-comptes de REQUIREMENTS.md (Open Q1). Flag al desenvolupador: si l'aula no és de confiança, cal un secret compartit mínim (fora d'abast v1). |
| T-01-SC | Tampering | npm installs (vite, lucide [SUS]) | high | mitigate | Checkpoint humà bloquejant (Task 1) verifica legitimitat a npmjs.com abans d'instal·lar; versions locked; cap postinstall detectat a RESEARCH.md. |
</threat_model>
