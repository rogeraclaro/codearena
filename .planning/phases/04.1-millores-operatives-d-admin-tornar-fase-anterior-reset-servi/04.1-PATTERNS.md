### `src/server/gameState.js` â€” add `previousPhase()` (model, transform)

**Analog:** `nextPhase()` at line 309 (exact mirror â€” decrement instead of increment).

**Core pattern to copy** (lines 299-314):
```javascript
function startPhase(phase, durationMs) {
  if (!PHASE_ORDER.includes(phase)) return false;
  if (!(Number.isFinite(durationMs) && durationMs > 0)) return false;
  state.phase = phase;
  state.phaseEndsAt = Date.now() + durationMs;
  state.timerStatus = 'running';
  state.remainingMsAtPause = null;   // D-02: timer nou, mai recupera pausa
  return true;
}

function nextPhase(durationMs) {
  const currentIndex = state.phase ? PHASE_ORDER.indexOf(state.phase) : -1;
  const nextIndex = currentIndex + 1;
  if (nextIndex >= PHASE_ORDER.length) return false;
  return startPhase(PHASE_ORDER[nextIndex], durationMs);
}
```
`PHASE_ORDER = ['html', 'css', 'js']` (line 24).

**New function** (mirror, `prevIndex < 0` guard â€” Pitfall 5):
```javascript
function previousPhase(durationMs) {
  const currentIndex = state.phase ? PHASE_ORDER.indexOf(state.phase) : -1;
  const prevIndex = currentIndex - 1;
  if (prevIndex < 0) return false;   // ja a la primera fase â€” no-op
  return startPhase(PHASE_ORDER[prevIndex], durationMs);
}
```

**Why `startPhase` satisfies D-02/D-03 for free:** it resets `remainingMsAtPause = null` (D-02: timer nou) and NEVER touches `team.doneAt` / placement / cssValues (D-03: equips congelats segueixen congelats). Do NOT add any `doneAt` reset in the prev-phase path.

**Export registration** (line 363, add to the `gameState` object literal next to `nextPhase`):
```javascript
export const gameState = {
  ...,
  startPhase,
  nextPhase,
  // add: previousPhase,
};
```
